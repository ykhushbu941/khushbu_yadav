# Clever Cubes

Clever Cubes is a responsive, web-based platform offering interactive, distraction-free puzzle games. This project provides a full-stack MVP featuring dynamic Sudoku, Word Search, and a Sliding Tile puzzle. The project documentation defines the product scope, required features, reference assets, build instructions, and evaluation rubrics.

## Project Overview
Clever Cubes is a responsive, web-based platform designed to provide a distraction-free environment for interactive puzzle games. The project aims to deliver a high-performance MVP that allows users to play Sudoku, Word Search, and Sliding Tile puzzles on any device. The focus is on secure user authentication, accurate game-logic algorithms, and a seamless, mobile-first user experience.

* **MongoDB**
* **Express.js**
* **React**

## Core Features
* **User registration and login** with secure authentication.
* **Puzzle generation** for Sudoku, Word Search, and Sliding Tile.
* **Dynamic Timer** for tracking match/game completion.
* **Real-time Input Validation** for puzzle moves.
* **Leaderboard and scoring** for tracking competitive performance.
* **User dashboard** for viewing puzzle history and personal stats.
* **Mobile-first design** with responsive scaling for all devices.
